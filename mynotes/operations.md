# Operasi di Python

## Arithmetics
   - \+ : penambahan
   - \- : pengurangan
   - \* : pengalian
   - \/ : pembagian
   - ** : pangkat
   - \// : 
   - \% : modulus

## Assignments
   - \= : Menetapkan nilai

## Comparison
   - ==: Sama dengan
   - \>: lebih besar
   - \>=: lebih besar sama dengan
